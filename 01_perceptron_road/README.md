# Персептрон для принятия решения: переходить дорогу или нет

## Задача

Этот проект представляет собой реализацию простейшего искусственного нейрона (персептрона) для бинарной классификации. Модель принимает решение, стоит ли пешеходу переходить дорогу, на основе трёх входных параметров.

**Входные данные:**
1.  Расстояние до приближающегося автомобиля (в метрах).
2.  Скорость автомобиля (в км/ч).
3.  Статус светофора (1 — горит зелёный, 0 — горит красный).

**Выходное значение:**
- `0` — переходить дорогу **нельзя** (опасно).
- `1` — переходить дорогу **можно** (безопасно).

## Реализация

Этот код написан на чистом Python с использованием библиотеки NumPy. Основная цель — продемонстрировать базовый принцип работы нейронных сетей: обучение подбором весов и последующее предсказание на основе взвешенной суммы.

### Ключевые элементы кода
- **Функция активации:** Используется ступенчатая функция (пороговая), которая возвращает 1, если взвешенная сумма превышает заданный порог, и 0 в противном случае.
- **Обучение (Дельта-правило):** Веса корректируются пропорционально ошибке на каждом шаге обучения. Это позволяет модели постепенно находить правильные значения весов для разделения входных данных на два класса.
- **Предсказание:** После завершения обучения модель может мгновенно выдать результат для новых, незнакомых данных.

## Как использовать

1.  Убедитесь, что у вас установлена библиотека NumPy.
2.  Запустите файл `perceptron.py`.
3.  В коде присутствует пример датасета для обучения. Вы можете заменить его на свои собственные данные.
4.  После обучения вы можете использовать функцию `predict()` для проверки работы модели на новых примерах.

## Результат

Обученный персептрон демонстрирует способность корректно определять безопасный момент для перехода. Например, модель учитывает, что даже при зелёном свете переходить дорогу опасно, если автомобиль находится слишком близко и движется с высокой скоростью.
