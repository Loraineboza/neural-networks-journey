# Нейросеть "IsCinema" (Пойдём ли мы в кино?)

## Описание задачи
Нейросеть предсказывает, стоит ли идти в кино, на основе трёх факторов:
- `x1` — хороший ли фильм? (1 — да, -1 — нет)
- `x2` — есть ли деньги? (1 — да, -1 — нет)
- `x3` — будет ли компания? (1 — да, -1 — нет)

## Архитектура сети
- Входной слой: 3 нейрона (признаки)
- Скрытый слой: 2 нейрона
- Выходной слой: 1 нейрон
- Функция активации: гиперболический тангенс (`tanh`)

## Обучение
- Оптимизатор: RMSprop (learning rate = 0.1)
- Функция потерь: L1Loss
- Количество итераций: 1000
- Тип обучения: стохастический градиентный спуск (выбор случайного примера на каждой итерации)

## Результаты
После обучения сеть выдаёт предсказания, близкие к правильным ответам:
- Для `1.0` → предсказание около `0.99`
- Для `-1.0` → предсказание около `-0.99`
- Ошибка: ~8.14e⁻⁵ (практически идеально)

## Структура проекта
- `IsCinema` — класс нейросети (наследуется от `nn.Module`)
- `main()` — функция, содержащая обучение и проверку
- `x_train` — обучающая выборка (8 примеров)
- `y_train` — правильные ответы
