# Папка activation_funcs

Эта папка содержит примеры функций активации, которые часто используются в нейронных сетях. Каждый файл показывает:

- как функция выглядит в виде математической формулы;
- как ее можно реализовать с помощью PyTorch;
- как она работает на примере входного значения (тензора).

## Содержимое

В папке представлены следующие функции активации:

- ReLU
- Leaky ReLU
- Sigmoid

- Softmax
- Softplus
- Swish
- ELU
- SELU
- GELU

## Для чего нужны функции активации

Функции активации вводят нелинейность в нейронные сети. Без них сеть по сути была бы линейной моделью, даже если в ней много слоёв.

Они позволяют модели:

- учиться сложным зависимостям;
- моделировать нелинейные закономерности;
- лучше работать с задачами классификации и регрессии.

## Краткое описание функций

### 1. ReLU
Формула:

$$f(x) = \max(0, x)$$

Это одна из самых простых и популярных функций активации.

### 2. Leaky ReLU
Формула:

$$f(x) = \max(\alpha x, x)$$

Позволяет избежать проблемы "мертвых" нейронов для отрицательных значений.

### 3. Sigmoid
Формула:

$$f(x) = \frac{1}{1 + e^{-x}}$$

Возвращает значения от 0 до 1.

### 4. Tanh
Формула:

$$f(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$$

Возвращает значения от -1 до 1.

### 5. Softmax
Формула:

$$f(x_i) = \frac{e^{x_i}}{\sum_j e^{x_j}}$$

Часто используется в последнем слое для многоклассовой классификации.

### 6. Softplus
Формула:

$$f(x) = \ln(1 + e^x)$$

Это гладкая аппроксимация ReLU.

### 7. Swish
Формула:

$$f(x) = x \cdot \sigma(x)$$

где

$$\sigma(x) = \frac{1}{1 + e^{-x}}$$

### 8. ELU
Формула:

$$f(x) = \begin{cases}
\alpha (e^x - 1), & x < 0 \\
x, & x \ge 0
\end{cases}$$

### 9. SELU
Формула:

$$f(x) = \lambda \begin{cases}
\alpha (e^x - 1), & x < 0 \\
x, & x \ge 0
\end{cases}$$

Используется в сетях с нормализацией и помогает поддерживать стабильность.

### 10. GELU
Формула:

$$f(x) = x \cdot \Phi(x)$$

где $\Phi(x)$ — функция стандартного нормального распределения.

Часто применяется в трансформерах.
